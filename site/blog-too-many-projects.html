反省

正直に言う。30個の70%完成より、3個の100%完成の方が価値がある。

1日で音声コマンド、パスワード管理、E2E暗号化、Beds24連携、ジオフェンス、ハウスマニュアル、収支レポート、申請書類生成、4段階チーム管理、オートロック解除、バックグラウンド予約通知を実装した。全部「動く」。でも全部「磨かれてない」。

サイトは8回リデザインした。8回やって分かったのは、1人のユーザーに見せてフィードバックをもらう方が、8回のリデザインより価値があるということ。

広く浅くから、狭く深くへ。

これからは、自分の物件で1週間使って、バグを全部潰して、1つずつリリースしていく。

おすすめ順に紹介

僕が実際に使っていて、今すぐ試してほしいものから順番に。

1
Koe — 声で入力
ローカルで動くWhisper音声認識。コードを声で書く。メールを声で送る。無料で高精度。仕事が10倍速くなった。毎日使ってる。
ダウンロード GitHub
2
KAGI — スマートホーム管理
自分の物件をこれで管理してる。SwitchBotの鍵、Hueの照明、Beds24の予約が1つのアプリに。E2E暗号化でゲストにシェアできる。
TestFlight GitHub
3
パシャ — レシート経費管理
レシートを撮るだけ。AIが自動で仕分け。確定申告の時期に地獄を見たくない人へ。電子帳簿保存法にも対応。
TestFlight GitHub
4
chatweb.ai — AIチャット
Rustで書いたAIチャットプラットフォーム。マルチモデル対応。ストリーミング。自分のAIアシスタントとして毎日使ってる。

ポン — 電子契約・署名
契約書にスマホで署名してそのまま送信。フリーランスの契約管理が楽になる。
TestFlight GitHub
6
チャリン — 収入・請求書
